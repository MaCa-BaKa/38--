# 38-日语学习小程序

**系统功能特点：**

##### 技术栈: springBoot+Vue+Mysql+Uni-app



##### 功能点: 用户端  管理员端



##### 用户端: 

1.首页: 轮播图展示、N2词汇列表、网站公告列表

2.学习模块: 有五十音图 词汇 语法

3.社区模块: 可进行发帖  也可查看帖子进行回复 可查看小组

4.我的: 

N2词汇(点击查看详情  可进行评论 语音学习 释义查看)

签到打卡

我的收藏查看

练习记录

认错词汇

在线练习进行试卷答题



##### 管理员端: 

1.个人中心: 个人信息查看和修改

2.用户管理: 用户列表展示以及增删改查

3.词汇单词模块: 列表及增删改查

4.签到打卡: 签到记录查看

5.试卷管理: 对试卷进行增删改查 以及 禁用/启用

6.试题管理: 列表及增删改查

7.系统管理: 

设置用户端轮播图  

网站公告的增删改查

8.在线练习:

练习记录查看

认错词汇查看

在线练习试卷答题
